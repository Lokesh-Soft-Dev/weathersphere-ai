# 🌦️ WeatherSphere AI

A futuristic AI-powered weather dashboard built using React, Vite, Framer Motion, Tailwind CSS, and OpenWeather API.

## 🌐 Live Demo : [weathersphere-ai.vercel.app](https://weathersphere-ai.vercel.app/)

## 🚀 Features

* Real-time weather information
* Dynamic weather-based backgrounds
* AI-inspired weather insights
* Glassmorphism UI design
* Responsive design for Mobile, Tablet, and Desktop
* Live temperature, humidity, wind speed, and feels-like data
* Smooth animations using Framer Motion
* Modern and futuristic user experience

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript
* Tailwind CSS
* Framer Motion
* Axios
* OpenWeather API

## ⚙️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/WeatherSphere-AI.git

cd WeatherSphere-AI

npm install

npm run dev
```

## 🔑 Environment Variables

Create a `.env` file and add:

```env
VITE_WEATHER_API_KEY=YOUR_API_KEY
```
